== Finding deals in Auckland {<img src="https://codeclimate.com/github/juanroldan1989/my-budget-app/badges/gpa.svg" />}[https://codeclimate.com/github/juanroldan1989/my-budget-app] {<img src="https://travis-ci.org/juanroldan1989/my-budget-app.svg?branch=master" alt="Build Status" />}[https://travis-ci.org/juanroldan1989/my-budget-app] {<img src="https://coveralls.io/repos/github/juanroldan1989/my-budget-app/badge.svg?branch=master" alt="Coverage Status" />}[https://coveralls.io/github/juanroldan1989/my-budget-app?branch=master]

Responsive web application built to look out single/combined deals in Auckland, filtering them by Price and Keywords ("tours", "hotels", "food", "drinks", "beaches" and "fun").

---------------------------------------------------------------

<b>Local Development</b>
* <code>git clone git@github.com:juanroldan1989/my-budget-app.git</code>
* <code>bundle install</code>
* <code>rake db:create db:migrate db:test:prepare</code>
* Import deals from http://bookme.co.nz and http://www.eventfinda.co.nz: <code>rake get:deals:websites</code>
* Combine deals by price: <code>rake deals:set:combined</code>
* <code>rspec spec</code>
* <code>foreman start</code>
